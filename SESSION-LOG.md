# SESSION-LOG — 進度 hub

← [CLAUDE.md](CLAUDE.md)｜[INDEX.md](INDEX.md)

跨工作流的**進度中樞**。事情告一段落、因應需求結束、或臨時中止時，把**還沒完成**的 in-flight / open 進度記在這裡。

**規範**：
- **只列 open**：完成即移除，不留已完成清單（歷史在 git log / commit 訊息）。
- 需要**使用者親自做/驗證**的（跨平台實測、外部環境、權限）不記這裡 → 記到 [WAIT_USER.md](WAIT_USER.md)。
- 單檔工作流的 open 進度直接列在這；資料夾型工作流若有自己的 `session-log.md`，這裡只放一行指標。
