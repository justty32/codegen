# 點子（idea）— 工作流入口

← [index](../index.md)｜[CLAUDE.md](../CLAUDE.md)

記一個**奇思妙想**——還**不確定要不要做**的東西。門檻最低：想到就丟進來，不必先評估可行性或排期。

## 用法

- 直接把點子追加到下面的清單，一條一行（一句話講清楚「想解決什麼 / 想加什麼」）。
- 某個點子**確定會做**（只是不知何時）→ 移到 [roadmap](roadmap.md)。
- 某個點子**要開始討論方案**→ 進 [spec](spec.md)。
- 不再想做的就刪掉。

規劃管線：**idea（本工作流）** → roadmap → spec → plan → build。

## 膨脹時

點子多到需要分類（依主題：CLI / 解析 / 跨語言 / 效能…）→ 升級成資料夾（`idea/ideas.md` 入口 + 各主題檔）。
